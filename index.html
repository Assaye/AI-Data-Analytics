About the Course

This 8-week guided course focuses on mastering no-code AI tools for task automation in work and research.

Websites, Documents & Presentations

How LLMs work to generate text, create project websites with no-code tools (like Canvas), and design clear presentations.

AI-Automated Data Analysis

Cursor and other AI tools like Code Interpreter and Julius to process datasets, visualize results, and create automated reports.

GenAI for Research & Text Analysis

Using AI for efficient literature reviews (Elicit, LiteRev), text classification, and synthesizing knowledge from large text collections.

Custom GPTs and Chatbots

LLM architecture (RAG systems) to build custom GPTs and interactive chatbots for research assistance.

AI for Task Automation

Strategies for automating repetitive tasks using AI agents, email response automations, and browser control.

GenAI for Communication

AI-driven approaches for research dissemination, including summaries, designing scientific figures, and video abstracts.

Capstone Project & Showcase

Collaborating to create and refine a final AI solution, tool, or presentation, concluding with a showcase to peers and guests.
