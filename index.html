About the Course

This 8-week guided course focuses on mastering no-code AI tools for task automation in work and research.

Module 1: Websites, Documents & Presentations

Learning how LLMs work to generate text, create project websites with no-code tools (like Canvas), and design clear presentations.

Module 2: AI-Automated Data Analysis

Mastering AI tools like Code Interpreter and Julius to process datasets, visualize results, and create automated reports.

Module 3: GenAI for Research & Text Analysis

Using AI for efficient literature reviews (Elicit, LiteRev), text classification, and synthesizing knowledge from large text collections.

Module 4: Custom GPTs and Chatbots

Diving into LLM architecture (RAG systems) to build custom GPTs and interactive chatbots for research assistance.

Module 5: AI for Task Automation

Discovering strategies for automating repetitive tasks using AI agents, email response automations, and browser control.

Module 6: GenAI for Communication

Exploring AI-driven approaches for research dissemination, including summaries, designing scientific figures, and video abstracts.

Module 7: Capstone Project & Showcase

Collaborating to create and refine a final AI solution, tool, or presentation, concluding with a showcase to peers and guests.
