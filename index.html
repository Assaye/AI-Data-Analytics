About the course

An 8‑week, live cohort bootcamp focused on no‑code tools and practical AI workflows for research productivity. Features include weekly live workshops, help sessions, a private community, personalized feedback, capstone coaching, and a shareable certificate with lifetime access to tutorials.

Module 1 — Websites, Documents & Presentations

How LLMs work; generate Markdown/LaTeX/HTML; build simple research sites; use AI presentation tools for clear scientific communication.

Module 2 — AI‑Automated Data Analysis

Use ChatGPT Code Interpreter; build automated dashboards; process datasets; batch file conversion; automated reports and spreadsheet workflows.

Module 3 — GenAI for Research & Text Analysis

Efficient literature reviews (Elicit, LiteRev); text classification; sentiment analysis; knowledge maps; summarization and synthesis systems.

Module 4 — Custom GPTs & Chatbots

LLM chat architecture, context windows, and RAG; custom GPTs; Botpress chatbots; knowledge base integration.

Module 5 — AI for Task Automation

AI automation and agent‑based systems; email response automations; browser control agents; orchestration of complex flows.

Module 6 — Communication & Outreach

AI‑generated summaries for multiple audiences; scientific figures and diagrams; video abstracts; prompts for visualization; content scheduling.

Module 7 — Capstone Project & Showcase

Project progress checks, peer feedback, final presentations, and a public Q&A.
